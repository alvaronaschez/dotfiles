# dotfiles

Backup of my dotfiles using [GNU Stow](https://www.gnu.org/software/stow/)

Inspired by the excelent article [How I manage my dotfiles using GNU Stow](https://tamerlan.dev/how-i-manage-my-dotfiles-using-gnu-stow/)
